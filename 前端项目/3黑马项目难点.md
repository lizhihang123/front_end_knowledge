# 2. 搜索框这个地方的功能怎么去设计



1. 有三块内容，一进入页面，没有输入任何内容，就显示“搜索历史记录”
2. 一旦输入内容，就显示“联想的一些建议”
3. 一旦点击这些建议，或者是敲回车，就会显示搜索的结果
4. 一旦点击结果，就会跳转到指定的文章



>1. 使用v-if和v-else-if v-else来控制
>
>2. 搜索框会有变量搜索值
>
>   还有一个结果变量控制结果组件是否显示，
>
>   如果搜索框一聚焦，就让结果变量为false，我们要展示联想建议，如果点击，就显示结果的组件
